## **Description**
Developed an Android application featuring text recognition, translation, object detection, and image-based information retrieval. The app uses Google ML Kit for text recognition and translation, and implements YOLOv11n with LiteRT for real-time object detection. It also combines ML Kit image labeling with YOLOv11 to extract object names and perform searches using the Google Custom Search API.

## **Software**
IDE: Android Studio

Program Language: Java
